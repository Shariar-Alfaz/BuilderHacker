# BuilderHacker.Core

Core functionality for BuilderHacker - a toolkit for builder pattern implementation.

## Overview
This package provides runtime builder capabilities and DI factory integrations.

### Key Components
- `EntityBuilder<T>`: A reflection-based runtime builder that can be used without source generation. Supports all .NET versions.
- `DefaultBuilderHackerFactory`: A DI-friendly factory implementation for resolving builders via `IServiceProvider`.

For more information and usage examples, please visit the [GitHub repository](https://github.com/Shariar-Alfaz/BuilderHacker).
