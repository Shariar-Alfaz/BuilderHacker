# BuilderHacker.Generator

Roslyn-based code generator for BuilderHacker - automatically generates builder pattern implementations for your classes.

## Overview
This package contains a high-performance incremental source generator that analyzes classes marked with `[GenerateBuilderHacker]` and automatically produces fluent builder implementations at compile time.

### Features
- Zero reflection overhead at runtime.
- Incremental generation for fast build times.
- Supports standalone and partial class builders.
- Correctly handles property inheritance and shadowing.

For more information and usage examples, please visit the [GitHub repository](https://github.com/Shariar-Alfaz/BuilderHacker).
